Session Improvement Plan Template

A reusable format for combining validation results from multiple skill frameworks
into a single actionable improvement plan.

================================================================================
FORMATTING CONVENTIONS
================================================================================

Character Set for Tables:
  ┌ ─ ┬ ┐    Top corners and horizontal
  │   │ │    Vertical separators
  ├ ─ ┼ ┤    Row separators with intersections
  └ ─ ┴ ┘    Bottom corners

Tree Structure for Session Flow:
  ├──        Branch (not last item)
  └──        Branch (last item)

Dividers:
  ---        Section separator (three dashes)

Text Formatting:
  ● Title    Bullet point for main title only
  Two-space  Indent all content after title line

Rationale Format:
  Why:       For 2 frameworks (single line)
  Why (N Frameworks):  For 3+ frameworks (bullet list)

================================================================================
USAGE GUIDELINES
================================================================================

When to Use This Template:
- After validating a session plan against 2+ skill frameworks
- When combining recommendations from different theoretical perspectives
- To create actionable improvement plans for coaches

Principles for Integration:
1. Find convergence first - Identify improvements that satisfy multiple frameworks
2. Prioritize by impact - Rank improvements by how many principles they address
3. Keep it actionable - Each improvement should have specific, implementable steps
4. Connect to theory - Always explain "Why" with reference to source principles
5. Preserve session intent - Improvements should enhance, not replace, original design

Adaptations for Different Framework Counts:
- 2 frameworks: Standard format, single-line "Why:" rationale
- 3+ frameworks: Add Framework Overview table, bullet-point "Why:" rationale
- Single framework: Simplify to just improvements without cross-referencing

Recommended Number of Improvements:
- Minimum: 3 improvements (covers main gaps)
- Typical: 5-8 improvements (comprehensive but digestible)
- Maximum: 10 improvements (beyond this, consider phased plans)

Table Column Alignment:
- Ensure columns align vertically for readability
- Use consistent spacing within columns
- Pad shorter text to maintain alignment
